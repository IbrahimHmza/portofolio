    * How To Add CSS :

  1. Inline : In this case we add css as an attribute in HTML tag
  2. Internal : In this case we add Style tage after head tag and add the selector that we need to style
  3. External :In this case we add CSS as a separate file and link it to our HTML file, we put the CSS link in element Link after head

    * CSS Selector:

  1. Element Selector
  2. Class Selector: begins with dot and we put it in html as attribute called 'class' =
  3. Id Selector: begins with hash we put it in html as attribute called 'id' =
    Note: the id must be unique in html file
  4. Attribute Selector: you will first put the element and the the attribute between[ ]
